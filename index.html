Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

Volunteers stepped forward to have the bacteria in their bellybuttons analyzed. This website contains a dashboard that explores the biodiversity in our bellybuttons, and it is interactive. By selecting the id number of the test subject in the dropdown list, information on the website is updated. Demographic information, including ethnicity, gender, age, and location can be found in a table below the dropdown list. The first chart is a bar chart of the Top 10 bacterial strains found in the subject's bellybutton and the second chart is a gauge showing that subjects bellybutton washing frequency. Finally, the last chart is a bubble chart showing the number of bacteria cultures per sample for the subject. Both the bar chart and the bubble chart will show additional information about the bacteria cultures when the mouse pointer hovers over the chart. Each chart updates as a new test subject is selected from the dropdown list.
